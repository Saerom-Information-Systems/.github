# Vue 프로젝트 설치 가이드

다음 프로젝트는 Vue 기반이며 동일한 설치 절차를 따릅니다.

## 대상 프로젝트

- vue_portal
- vue_approval
- vue_search
- vue_bbs
- vue_teamspace
- vue_mobile
- vue_manual
- vue_video
- vue_pms
- vue_calendar

## 설치 절차

1. 저장소 클론
    ```bash
    git clone <레포지토리 주소>
    ```

2. 디렉토리 이동
    ```bash
    cd <프로젝트명>
    ```

3. 패키지 설치
    ```bash
    npm install
    ```

4. 환경변수 파일 `.env` 생성

5. 로컬 서버 실행
    ```bash
    npm run serve
    ```
